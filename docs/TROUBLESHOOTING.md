# Troubleshooting

- **Compilation says undeclared symbol:** add every axiom, predecessor, context,
  successor, and command module to the alphabet.
- **Different plants:** compare model hash, package version, seed, parameters,
  time, limits, registry implementations, and descriptor version.
- **Truncated result:** inspect `LIMIT_*` diagnostics and statistics before
  increasing a budget.
- **Part pool exhausted:** lower LOD or raise the explicit pool/Instance limit.
- **EditableMesh unavailable:** verify capability injection, experience security
  settings, permissions, memory budget, and current platform docs.
- **Frame stall:** use incremental derivation, streamed rendering, and lower
  geometry/organ density.

## [0.1.0] - 2026-07-14

### Added

- Bounded deterministic L-system, graph rewriting, turtle, topology, botanical,
  geometry, phyllotaxis, timed development, cellular, and IFS modules.
- Versioned model specs, registries, serialization, caching, incremental sessions,
  presets, diagnostics, LOD, pooling, Part and EditableMesh adapters.
- Tests, examples, benchmark suite, full guides, generated API site, CI, Pages,
  release automation, and clean consumer/pack verification.

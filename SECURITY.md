# Security policy

Security fixes are supported for the latest published minor version. Please do
not open a public issue for a vulnerability. Use GitHub's private vulnerability
reporting for `mateusdcc/plant-generator`; include affected versions, impact,
reproduction, and any suggested mitigation.

Do not submit credentials, Roblox cookies, npm tokens, or private place data.
See [docs/SECURITY.md](docs/SECURITY.md) for safe model/descriptor handling.
